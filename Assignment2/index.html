Reddit’s main slogan is “the frontpage of the internet”. This frontpage is supplied by reddit users and unlike social networking sites, all of the posted content is somehow accessible. Almost every moment in time an user posts something on reddit. Not everything is posted on 1 page, reddit is subdivided in subreddits (branches) which all contribute to certain topic or niche. It seems to be difficult when searching a subreddit which delivers to your specific needs. This is the first reason why I want to redesign reddit.

The second reason is that the computer interface is overwhelming. I did not count them, but there are a lot of clickable things when you load the website. Even I prefer to use the more user friendly interface on the mobile platform

In my version of reddit the search function is the most central element. Unlike the existing reddit search function which only redirects users to posts containing the search query, my search engine gives a better and more complete overview. With my method it should be easier to discover new unknown subreddits. As for the interface I do like to blur the line between the desktop and mobile version. Thus making it less text dependent and more image/icon dependent.

Reddit is an infinite scrolling page made to constantly deliver content. This element still is present in my re-design. The interface is accessible on all platforms because of the Javascript and CSS files.

Reddit is huge, and with my project I only covered a very little piece of it. But nevertheless I think this approach is viable and a good starting point for a total redesign of reddit.
